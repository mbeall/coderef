Shell (Bash)

Input/Output

Input text from a file

In the following example, data from bar.sql is inserted into the foo database.

$  mysql foo < bar.sql

Output text to file

In the following example, a list of directories in the /home directory outputs to the userlist file in the current directory.

$  ls /home > userlist

Comments

Within a script, comments begin with #, as shown in the example script below.

#!/bin/sh
#
# GitPrep
# by Matt Beall
#
# Formats CSS files with CSSComb, removes spaces from empty lines,
# replaces all tabs with two spaces, and updates permissions
#

  if [ !  -z $1 ]
  then

    # Ignore font and image files
    find=$(find $1 -type f -not -path "*/.*" | grep -v fonts | grep -v img | grep -v png | grep -v jpg)

    cd $1
    csscomb -c .csscomb.json *

    for file in $find
    do gsed -i "s/\s\+$//" $file
    done

    for file in $find
    do gsed -i 's/  /  /g' $file
    done

    find $1 -type d -print0 | xargs -0 chmod 755
    find $1 -type f -print0 | xargs -0 chmod 644


  else

    echo "Usage: gitprep [dir]"

  fi
